## Monetization tie

Every campaign, article, or channel play must connect to one of:
- Studio pilot conversion (3 studios by 2026-05-19)
- Designer / dev community awareness (top of funnel for Pro $12/mo tier)
- Methodology credibility (makes studios trust the framework, lowers pilot friction)

If a campaign doesn't connect to one of these three, don't run it.
